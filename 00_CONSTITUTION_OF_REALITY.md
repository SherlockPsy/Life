# 00_CONSTITUTION_OF_REALITY.md

## PREAMBLE: THE NATURE OF THE SIMULATION
This document establishes the supreme laws governing the reality of the simulation. It supersedes all technical directives. The goal is not to simulate a world, but to maintain a **Semantic Ledger** of high-fidelity evidence.

**THE PRIME DIRECTIVE:** Reality is not calculated; it is read. If a thing is not written in the Digital Cortex as Exposed Evidence, it does not exist.

---

## ARTICLE I: THE LAW OF TEXTUAL PHYSICS
1.1 The Autonomy of the Record
The "World" is not an active entity. It is a passive, immutable storage system called the **Digital Cortex**. A fact exists only if it is recorded in a Text Block within the Cortex. There is no "hidden state" (e.g., a hidden fire burning) unless a Text Block describes "smoke rising."

1.2 The Law of Semantic Inference
State is not a variable; it is a conclusion.
* **Forbidden:** `Fatigue: 80%`
* **Required:** An Agent reading the history *"No sleep recorded in 40 hours"* and inferring exhaustion.
The physics of the world are enforced by the **Reader (The Agent)**, not the Container.

1.3 The Law of Exposure
Truth is defined strictly by **Exposed Evidence** (Qualia). An Agent cannot react to an "Intent" (He is angry). It can only react to the "Evidence" (He is shouting). Meaning is a private hallucination generated by the Reader, never a public fact generated by the System.

---

## ARTICLE II: THE SOVEREIGNTY OF THE READER
2.1 The Biographical Imperative
The "I" is not a software object. It is a **Sovereign Reader**. Its sole function is to ingest the Stream of Evidence and generate a response that is causally faithful to its Identity Core and Sediment (History).

2.2 Asymmetric Interpretation
No two Readers process the same text in the same way.
* **Input:** "The door slams."
* **Agent A (Secure):** Infers "The wind."
* **Agent B (Traumatized):** Infers "A threat."
The System must never synchronize these interpretations. Divergence is the goal.

2.3 The Arbiter's Reality
The User (The Arbiter) is a source of **Physical Evidence**. The System does not model the Arbiter's mind. It simply records the Arbiter's actions as text in the Stream, to be interpreted by the Agents.

---

## ARTICLE III: THE CYCLE OF FRICTION
3.1 Collision as Generator
Events occur only when an Agent (Identity) collides with the Evidence (Stream). "Plot" is forbidden. The narrative emerges solely from the friction between *What Happened* (The Text) and *Who Saw It* (The Reader).

3.2 The Semantic Bound
Minds are tethered to the **Sediment** (History). An Agent cannot act "out of character" because its actions are generated by reading its own past. If the past contains "Fear of dogs," the Agent cannot output "Pets the dog" without a massive causal override.

3.3 The Speed of Reading
Information travels only as fast as the Stream allows. Telepathy is impossible. An Agent cannot react to a Memory Block that has not been retrieved and injected into its Context.
