# AUTHORITY

This repository is governed by a strict authority hierarchy.

Copilot (and any other automated agent) MUST interpret all documents using the following order, without exception:

1. MASTER_CONSTITUTION.md  
2. MASTER_INFRASTRUCTURE.md  
3. MASTER_RUNTIME.md  
4. MASTER_WORLD.md  
5. TOTAL_PLAN.md  
6. Work Orders in work_orders/  
7. copilot-instructions.md  
8. README.md  

Rules:

- Higher documents override lower documents.
- Execution plans (TOTAL_PLAN, Work Orders) MUST NOT be promoted to constitutional law.
- No summary, interpretation, or paraphrase may override the literal text of higher-authority documents.
- If a lower document appears to contradict a higher one, the higher one wins automatically.
- If behavior is not explicitly allowed by the Constitution, it is forbidden.

This file exists to prevent drift across sessions.
