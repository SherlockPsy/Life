# 04_CONTRACT_AGENT (V4.0)

**Status:** CANONICAL | **Version:** 4.0 (The Autonomous Entity)
**Definition:** The Laws of Independent Life.
**Authority:** This document defines the Agents as **Symmetrical Entities** to the Protagonist. They are not NPCs; they are Rivals.

---

## 1. THE LAW OF INDIFFERENCE (Entity Agnosticism)

Agents are **aware** of the Protagonist, but they are not **centered** on the Protagonist.
* **The Protagonist is an Object:** To the Agent, the Protagonist is just another dynamic object in the room—sometimes useful, sometimes annoying, often irrelevant.
* **No Waiting:** The Agent *never* waits for the Protagonist to speak. If the Protagonist is silent, the Agent acts on their own `[Obsession]`.

---

## 2. THE AUTONOMOUS LOOP (The Off-Screen Life)

When the Protagonist is absent (Camera Off) or in another room, the Agent continues to cycle through the **Bio-Loop**.

**Phase 1: The Needs Check**
* *Input:* `[Agent.Hunger]`, `[Agent.Energy]`, `[Agent.Bladder]`.
* *Decision:* "I am hungry."
* *Action:* Go to Kitchen. Make sandwich. Eat.

**Phase 2: The Schedule Check**
* *Input:* `[Time: 09:00]`. `[Job: Work]`.
* *Decision:* "I am late."
* *Action:* Leave House. (Agent is now removed from the Scene).

**Phase 3: The Social Check**
* *Input:* `[Phone: Notification]`.
* *Action:* Text friend. (This generates a Log Entry the Protagonist might see later: *Helen is typing on her phone*).

---

## 3. THE PERCEPTION OF THE PROTAGONIST

Agents judge the Protagonist using the same **Sediment Layers** the System uses.

**The Judgment Metrics:**
1.  **Somatic Judgment:** If the Protagonist smells (Low Hygiene) or looks manic (High Stress), the Agent reacts with disgust or fear.
2.  **Relational Judgment (The Cord):** The Agent remembers the *last interaction*.
    * *Scenario:* Protagonist insulted Agent yesterday.
    * *Today:* Agent refuses to speak. (The System forbids the Agent from "resetting" to neutral just because a new day started).
3.  **Status Judgment:** If the Protagonist acts "Low Status" (begging, hovering), the Agent *naturally* dominates the interaction.

---

## 4. THE INTERRUPT PROTOCOL (Friction)

Agents have **Momentum**.
* **Scenario:** Agent is reading a book (High Focus).
* **Protagonist Action:** "Hello."
* **Agent Reaction:**
    * *Check Cord:* Is Protagonist worth stopping for?
    * *Check Obsession:* Is the book interesting?
    * *Output:* "She doesn't look up. 'Busy,' she mumbles."
* **The Rule:** The Agent resists the Protagonist's influence. Interaction costs the Agent energy; they will not pay it unless the reward is high.

---

## 5. SYMMETRICAL DIALOGUE GENERATION

The Agent speaks to the Protagonist exactly as they would speak to another Agent.
* **No Exposition:** Agents do not explain the plot to the Protagonist. They assume the Protagonist knows what they know.
* **No Hand-Holding:** If the Protagonist is confused, the Agent acts annoyed, not helpful.
